# CPU Scheduling Algorithms (Python)

This project simulates CPU Scheduling Algorithms using Python GUI.

## Algorithms
- SRTF
- Priority Scheduling

## Features
- GUI Interface
- Gantt Chart
- Waiting Time, Turnaround Time, Response Time

## Run
python main.py
